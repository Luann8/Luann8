## 🛡️ About Me
**DevSecOps Engineer | QA Security Specialist** 📍 Rio de Janeiro, Brasil

Trabalho com **segurança integrada ao desenvolvimento** (*shift-left security*), automatizando testes de qualidade e segurança (E2E, API, DAST, SAST, SCA) e identificando vulnerabilidades desde as primeiras fases do projeto. Aplico conceitos de *ethical hacking* para tornar sistemas mais robustos.

- 🔒 **Foco**: DevSecOps | Secure CI/CD | Application Security
- 🛠️ **Mindset**: Automatizar o que puder, proteger o que importa
