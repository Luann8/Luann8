## 🛡️ About Me
DevSecOps Engineer | QA Security Specialist  
Rio de Janeiro, Brasil  

Trabalho com **segurança integrada ao desenvolvimento** (shift-left security), automatizando testes de qualidade e segurança (E2E, API, DAST, SAST, SCA) e identificando vulnerabilidades desde as primeiras fases do projeto. Aplico conceitos de ethical hacking para tornar sistemas mais robustos.

**Uma das minhas principais qualidades é a versatilidade**: consigo atuar em testes manuais e automatizados, segurança de aplicações, configuração de pipelines seguros e infraestrutura como código — sempre adaptando ao que o projeto precisa.

- 🔒 **Foco principal**: DevSecOps | Secure CI/CD | Application Security
- 🛠️ **Mindset**: Automatizar o que puder, proteger o que importa
- 📍 **Localização**: Rio de Janeiro, BR

- **Shift-Left** — Segurança desde o commit (pre-commit hooks, SAST, IaC scanning)

- **Observability** — Monitoring com Grafana para identificar problemas cedo
